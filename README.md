# Task API

## How to Run Locally
1. Clone repo
2. Run `npm install`
3. Start server: `npm run dev`
4. API runs on `http://localhost:3000`

## Endpoints
- POST /api/tasks → create a new task  
- GET /api/tasks → fetch all tasks
